# MovieApp
Student: Ana Bogdanova

## Purpose
MovieApp is a simple console application that allows users to:  
1. Search for movies by name using an external API  
2. Add movies to a favorites list and view the list 
3. Post comments on favorite movies
4. Clear the list of favorite movies

# How to Run the App

## Prerequisites
- .NET SDK 7.0 or later installed on your machine  

## How to Build and Run

1. Clone or download the repository 
2. Open the project in a terminal or VS Code
3. Run the API project  
dotnet run --project MovieApp.Api  
4. Run the Console app  
dotnet run --project MovieApp.Console  
5. Use the console app to navigate through the menu
6. Enjoy!
